# Comrades Design Stub

Comrades is post-MVP.

It will support real friend relationships, direct and group chat, verification, blocking, revocation, and owner-controlled low-priority shared compute.

Comrade queues are owner-authoritative queues whose resource owner selects eligible devices, comrades, workload classes, models, priorities, schedules, quotas, concurrency, retention, audit, and revocation rules.

Comrade access never grants project files, secrets, arbitrary tools, shell access, administration, or unrelated cluster state by default.
